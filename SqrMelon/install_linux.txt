Ubuntu
======

Install dependencies

   sudo apt-get install python-qt4-gl
   pip install -r requirements.txt

Run the project

    python SqrMelon/__init__.py


Arch Linux
==========

Install dependencies

    sudo pacman -S python2-pyqt4
    sudo pacman -S python2-sip
    sudo pacman -S python2-opengl

Run the project

    python SqrMelon/__init__.py
